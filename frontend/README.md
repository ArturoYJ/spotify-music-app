# Spotify Music Player - Angular

Aplicación web desarrollada en Angular para consumir la API de Spotify y mostrar información de música. Este proyecto académico permite explorar artistas, álbumes y canciones, practicando la integración de APIs REST en aplicaciones front-end modernas.

## Objetivo del Proyecto

Desarrollar habilidades en:
- Consumo de APIs REST (Spotify Web API)
- Gestión de autenticación OAuth 2.0
- Manejo de servicios HTTP en Angular
- Enrutamiento y navegación en SPA
- Diseño de interfaces responsivas con Angular v20

## Tecnologías Utilizadas

- **Angular**: 20.3.4
- **TypeScript**: Lenguaje principal
- **SCSS**: Estilos y diseño responsivo
- **Spotify Web API**: Fuente de datos musicales
- **Angular CLI**: Herramientas de desarrollo

## Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (versión 18 o superior)
- [npm](https://www.npmjs.com/) (viene con Node)
- [Angular CLI](https://angular.dev/tools/cli): `npm install -g @angular/cli`
- Credenciales de la API de Spotify ([obtenerlas aquí](https://developer.spotify.com/dashboard))
